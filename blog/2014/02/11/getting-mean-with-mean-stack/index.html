Getting Mean With Mean Stack

Feb 11th, 2014

It has been a busy week for me where I had been trying to setup a node js development/test environment where I can test my sample applications and learn node js. This environment will also act as a sandbox for me to learn node, mongo db and angular js. The whole process of finding a VPS (Amazon AWS), custom Linux image (Bitnami) and an appropriate framework was difficult. The point in favour of mean stack was the full set of node js libraries that would have helped me in developing a fully functional website. Along with this using yoeman automation tools I am sure I will be able to automate a large part of the developemnt process. Next I need to look into my deployment, process management, scheduling, testing and scaling process. In all this the quest form my perfect application is still on.

Tools/Frameworks used so far:

  1. Bitmani Node stack and bitnami cloud hosting (https://app.bitnamihosting.com/)

  2. Mean stack (refer http://mean.io)

  3. git-deploy (Easy way to deploy any code to sandbox/prod environment)

  4. pm2 (A very good cli tool for process management for node js applications https://github.com/Unitech/pm2)

Still there is a lot of way to go to really making a production ready application. As part of my app I am planning to integrate a blog as well so that I can track my development journey through out the way.
